What it does

  • Create or import custom item lists
  • Draw random results from your pool
  • Display results cleanly for group viewing
  • Reset and reuse instantly
  • Customize visuals with themes, boxes, and backgrounds
  • Use locally without accounts or cloud setup

Which version should I use?

LootBox is the game-focused version. Use it for RPG loot, treasure pulls, random rewards, campaign tools, and themed game-night draws.

LootBox Events is the general-purpose event version. Use it for raffles, classroom picks, prize drawings, office events, giveaways, and facilitator-led selections.

Both versions follow the same simple idea: load a list, open the box, draw a result.

How it works

  1. Load or import a list of items
  2. Open the box to draw a random result
  3. Take or leave items as needed
  4. Reset and reuse the list for the next session

Designed for live use at a table, on a screen, or in a shared session.

Privacy-first by design

LootBox is designed as a local-first app. It does not require an account, login, or hosted user profile.

FAQ

Do I need an account?

No. LootBox is designed to work without accounts or identity management.

What is the difference between LootBox and LootBox Events?

LootBox is themed for games and tabletop use. LootBox Events is themed for raffles, giveaways, classrooms, meetings, and general event use.

Can I use my own item lists?

Yes. LootBox is built around custom lists that can be reused, reset, and drawn from during a session.

Is the web version available?

Not yet. The web download is planned and will be added here when available.

Is an iPhone version available?

Not yet. Apple App Store availability is planned for a future release.

Can I use LootBox for non-game raffles?

Yes. LootBox Events is the best fit for raffles, prize drawings, classroom picks, office events, and giveaways.

Does LootBox store my data online?

No account or cloud profile is required. See the Privacy Policy for the current privacy details.
