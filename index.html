LootBox

A lightweight, local-first random draw app for dungeon masters, facilitators, and shared table experiences.

Web Version (Coming Soon) Privacy Policy

No account required. Runs locally. Built for quick setup and use.

What it does

  • Create or import custom item lists
  • Draw random results from your pool
  • Display results cleanly for group viewing
  • Reset and reuse instantly
  • Customize visuals with themes and styles

How it works

  1. Load or import a list of items
  2. Open the box to draw a random result
  3. Take or leave items as needed

Designed for live use at a table, on a screen, or in a shared session.

Availability

  • Android APK – available
  • Windows desktop build – available
  • Web version – in development
